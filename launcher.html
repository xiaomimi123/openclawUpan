一键安装 微信插件
安装后扫码连接微信，让 AI 在你的聊天中自动回复消息。建议使用小号，避免主号风险。
1
点击"开始安装"，自动下载微信插件
2
安装完成后，在 openclaw CLI 里执行启动命令显示二维码
3
用手机微信扫码即可连接，完成后即可对话
在微信里 @AI 或私信即可对话。断开后需重新扫码。所有对话按 Token 消耗从账户余额扣费。
已内置
Telegram 已内置
OpenClaw 原生支持 Telegram，无需额外安装插件。按下面步骤配置 Bot Token 即可使用。
1
在 Telegram 里找 @BotFather，发 /newbot 创建一个 Bot
2
把 BotFather 返回的 Bot Token 复制到 openclaw.json 的 telegram.botToken 字段
3
重启 Gateway，在 Telegram 里和你的 Bot 对话即可
Bot Token 是敏感凭据，请勿截图分享。如不慎泄露，立即在 @BotFather 用 /revoke 吊销。

飞书插件
在飞书应用里调用 AI，支持群聊 @ 与私聊。需要在飞书开放平台创建自建应用并拿到 App ID / App Secret。
1
点击"开始安装"，自动下载并运行飞书 onboard-cli
2
按终端提示填入 App ID / App Secret（飞书开放平台可查）
3
完成后在飞书应用里即可 @ 你的 AI 对话
安装过程会打开一个终端窗口用于输入配置。如果没弹出，检查系统是否允许启动器开子窗口。
